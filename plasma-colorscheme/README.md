### Installation

1. Move `Linen.colors` to `~/.local/share/color-schemes`
2. Launch System Settings > Appearance > Colors > Select Linen > Click Apply.
3. In System Settings, go to Appearance > Global Theme > Select Breeze Light > Click Apply.

That's it, enjoy!
